HTML, or Hypertext Markup Language, is a standard markup language cotaining information on websites fetched by the browser. The elements described by an HTML document form the building blocks of a web page. Your browser does not directly, show the contents of HTML, rather it uses the information within HTML to render the appropriate elements. On this page, the <b> tag is used to make the title at the top even more bold. A more complex tag that was used on this page was <script>. This was used for the functionality of the hamburger menu near the top-left of the page, so that it could animate and display the navigation menu.
