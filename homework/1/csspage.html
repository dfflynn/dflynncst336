CSS is the language which transforms the descriptive HTML document, styles it, and makes it look how the programmer wants it to. Where HTML says that an image exists, CSS says how big it is, or if it has a border. In this page, there exists the transform property, which is used along with scripting to perform animations on the web page. This was used to create the hamburger menu animations upon being clicked. On this page there also exists the word-wrap and white-space properties. These are used to properly display text wrapping for the paragraphs you are reading right now. The reason why there are two of them (used three times in the code), is due to the fact that different browsers display certain elements differently. Learn more about these differences on the Browsers page.
